Terminology

Internet

The Internet is a large network that links computers all over the world. People may exchange information and interact through the Internet from any location that has an Internet connection. [1]

World Wide Web

The World Wide Web (WWW), sometimes known as the Web, is the Internet's primary information retrieval service (the worldwide computer network). The Web provides users with access to a huge array of documents that are linked to one another via hypertext or hypermedia links, electronic connections that link similar bits of information to provide a user quick access to them.[2]

Hypertext

A hyperlink is an electronic connection that connects two or more pieces of information to provide a user quick access to them. A hypertext, or text-based hyperlink, is a feature of some computer systems that allows a user of electronic media to choose a word from text and obtain further information about that word, such as a definition or references within the text.[3]

Multimedia

Multimedia is a phrase used to represent two or more forms of media merged into a single package, generally signifying a mix of any or all of the following: video, sound, animation, text, and images. The usage of multimedia allows the user to affect the display of information.[4]

CLI

A Command-line interface (CLI) An interactive system that accepts user input in the form of lines of text. These instructions are taught to the user by reading an online help system or a reference handbook. Users who are already familiar with the interface can utilize acronyms or mnemonic instructions to speed up access and minimize the amount of keystrokes necessary for a specific command.[5]

GUI

A graphical user interface (GUI) is a computer environment that facilitates user interaction by expressing programs, instructions, files, and other choices as visual components like as icons, pull-down menus, buttons, scroll bars, windows, and dialog boxes. The user may commence several operations, such as opening a program or printing a document, by selecting one of these graphical components, either with a mouse or via a menu.[6]

HCI

A human-computer interface (HCI), a communication between a human user and a computer system, namely the usage of input/output devices and related software. Graphics gadgets, touch-sensitive devices, and voice-input devices are examples. They must be designed in such a way that a human and a computer may communicate in an efficient and desired manner.[7]

SFTP

SFTP (SSH File Transfer Protocol) is a file transfer protocol that is secure. It operates via the SSH protocol. It provides complete SSH security and authentication capability. As a file transfer protocol, SFTP has mostly superseded classic FTP and is rapidly overtaking FTP/S.[8]

SSH

SSH, or Secure Shell, is a network communication protocol that allows two computers to connect (c.f. http, or hypertext transfer protocol, which is the protocol used to send hypertext such as web pages) and exchange data. Because the communication between the two computers is encrypted, it is acceptable for usage over unsecured networks.[9]

telnet

Telnet is a networking protocol that allows you to remotely access a computer system. Remote connections between various computer kinds were required by computer users. [10]

HTTP

HTTP, or HyperText Transfer Protocol, is a common application-level protocol used on the World Wide Web to exchange files. HTTP is a protocol that operates on top of the TCP/IP protocol and (eventually) the QUIC protocol.[11]

HTTPS

Hypertext Transfer Protocol Secure (HTTPS) is a variation of the conventional web transmission protocol (HTTP) that adds a layer of security to data in transit via an SSL or TLS protocol connection. HTTPS offers secure communication and encrypted communication between a distant user and the principal web server.[12]

HTML

HTML, or hypertext markup language, is a formatting method for presenting information obtained through the Internet. Each retrieval unit is referred to as a Web page (from the World Wide Web), and such pages usually include hypertext connections that enable the retrieval of related sites. HTML is a markup language used to create Web pages.[13]

CSS

CSS is an abbreviation for "Cascading Style Sheet." The layout of Web pages is formatted using Cascading Style Sheets. They can be used to specify font styles, table sizes, and other features of Web pages that were previously solely described in the HTML.[14]
